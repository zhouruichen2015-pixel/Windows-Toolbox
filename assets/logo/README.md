| zrc_logo_256x256.png | 256x256 | PNG | 中等尺寸 |
| zrc_logo_128x128.png | 128x128 | PNG | 小尺寸 |
| zrc_logo_64x64.png | 64x64 | PNG | 图标尺寸 |

| zrc_logo_monochrome.png | 512x512 | PNG | 单色版本 |

## 注意

**占位符说明**：当前目录下的Logo文件为占位符，实际使用时需要：

1. 使用设计软件（如Figma、Photoshop、Illustrator等）制作Logo
2. 按照上述规格导出不同尺寸的文件
3. 替换本目录下的占位符文件

## 临时方案

在正式Logo设计完成前，可以临时使用原MHY工具箱的Logo，或使用简单的文字Logo。
